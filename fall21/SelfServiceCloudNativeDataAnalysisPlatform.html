Project Overview:

Background:  Two Sigma is a financial science company which invests heavily into data science. In order to enable our scientists to conduct research, we have to build and maintain massive-scale platforms for their work. Cloud computing has allowed us to become more agile, cost-efficient, and scale when providing those platforms.

With the ever-evolving cloud, managed services (SaaS) have become an extremely useful and important tool to the cloud toolkit. When choosing to use a managed service versus build your own, there can and will be trade offs. The goal of this project is to build a solution without using managed services, compare the differences and explain the trade offs.

Project Specifics: The goal of this project is to build a cloud data ingestion and analysis platform, capable of letting a select group of arbitrary users work with datasets quickly and efficiently. You will have to learn how to manage cloud resources, work with modern ETL (Extract/Transform/Load) pipelines, handle storage of this data, and also gain the skills to then analyze this data. Usually, one must decide what is needed and how to allocate each piece.
