Scaling remote sensing data processing in the Cloud with Ray

Project Logistics:

  • Mentors: John Liagouris (liagos@bu.edu)
  • Will the project be open source: Yes

 

Past Experience:

  • Strong Python programming skills
  • Previous experience with Ray and Python libraries, such as Dask, xarray, and numpy is a plus but not required to work in this project

Project Overview:

Background: The data collected by satellites through a process known as remote sensing helps researchers study the local effects of climate change on ecosystems all over the world.

To create effective policies that protect the Earth’s climate, researchers need to bett­er understand how the Earth’s land cover, or the physical makeup of the land, changes over time. However, the amount of collected data is enormous and researchers need scalable systems to seamlessly distribute these data across large clusters of machines and perform the analysis in parallel.

 

Project Specifics: In this project students will use Ray to implement a scalable Python application for remote sensing data processing in the cloud. The application will download real satellite image data from web archives, transform the raw data into time series, and apply a given regression model to identify land cover changes over large periods of time.

Some Technologies you will learn/use:

  • Ray and its computational model (tasks, actors)
  • How to deploy applications in the cloud using VMs and OpenShift containers
  • Popular Python libraries, including Dask, xarray, and numpy
  • How to profile the performance of distributed Python applications
